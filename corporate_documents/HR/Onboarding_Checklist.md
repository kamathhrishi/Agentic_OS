# New Employee Onboarding Checklist

## Pre-Start (Week Before)
- [ ] Send welcome email with first day details
- [ ] Prepare workstation and equipment
- [ ] Set up IT accounts (email, systems access)
- [ ] Prepare onboarding materials
- [ ] Schedule first week meetings

## Day 1
- [ ] Welcome meeting with manager
- [ ] Office tour and introductions
- [ ] IT setup and system access
- [ ] Review employee handbook
- [ ] Complete required paperwork
  - [ ] Employment contract
  - [ ] Tax forms
  - [ ] Benefits enrollment
  - [ ] Emergency contact form
- [ ] HR orientation session

## Week 1
- [ ] Department orientation
- [ ] Team introductions
- [ ] Review job description and expectations
- [ ] Access to required tools and systems
- [ ] Security training
- [ ] Compliance training
- [ ] First assignment/project

## Month 1
- [ ] Weekly check-ins with manager
- [ ] Complete all required training
- [ ] Set up performance goals
- [ ] Meet with key stakeholders
- [ ] Review progress and feedback

## Month 2-3
- [ ] Mid-point check-in
- [ ] Adjust goals if needed
- [ ] Complete any remaining training
- [ ] Full project ownership
